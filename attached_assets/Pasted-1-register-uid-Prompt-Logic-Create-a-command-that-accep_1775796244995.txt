3. /profile
Prompt/Logic: Fetch the player object. Display:

Adventure Rank, Achievements, Signature.

Spiral Abyss: towerFloorIndex-towerLevelIndex.

Imaginarium Theater: theaterFloorIndex and theaterStarCount.

Showcase: A list of the 8 characters currently in the user's in-game profile.

4. /list [category]
Prompt/Logic: A command with a choice of characters or artifacts.

If characters: List all 8 showcase characters with their levels and Crit Value (CV).

If artifacts: List the top 5 artifacts equipped on the current showcase characters, sorted by CV.

CV Calculation: (CritRate×2)+CritDMG.

5. /build [character_name] [hide_details]
Prompt/Logic: * character_name: Use a String option with autocomplete for the 8 characters in their showcase.

hide_details: A Boolean (True/False).

Action: If hide_details is True, use Canvas to redact/blur the UID and the In-Game Name on the final image. The image should mimic the Akasha/Enka layout showing artifacts, stats, and the character card.

6. /cv_calc [crit_rate] [crit_dmg]
Prompt/Logic: Simple math command.

Calculation: CV=(crit_rate×2)+crit_dmg

Response: "Your artifact has a [CV] CV. (Rating: [Fledgling/Good/Great/Godly])"

Prompt/Logic: Scans all 8 characters in the showcase. Extracts every artifact. Sorts them by CV.

Output: An embed showing the "Hall of Fame" for the user's gear.

Flower: 45.2 CV (Raiden Shogun)
